# 示例 Prompt

这些 prompt 用于展示 `learning-coach` 的典型触发方式。重点不是“问知识”，而是把学习推进到一个真实任务。

## 1. 技术学习

```text
我有一点编程基础，想学 TypeScript / React / AI Agent，但看教程总是学完不会用。别给我完整路线，帮我设计今天能完成的第一个小闭环。
```

## 2. 英语口语

```text
我想练英语口语，能看懂文章，但一开口就卡。请给我一个 60 分钟能完成的真实表达练习，并告诉我失败时该反馈什么。
```

## 3. 备考

```text
我准备一个重要考试，刷题很多但分数不上去。帮我诊断是不是复盘方式有问题，并安排下一次 90 分钟复习。
```

## 4. 写作和短视频表达

```text
我想提升写作和短视频表达，总是收藏爆款拆解，但自己写出来很散。帮我把一个选题拆成今天能交付的脚本片段。
```

## 5. 产品、运营或研究方法

```text
我想学产品分析和用户研究，但不知道怎么把方法用到真实问题里。请帮我设计一个最小研究任务，不要给我一堆书单。
```

## 6. 目标太多

```text
我同时想学英语、写作、AI、剪辑和产品分析，但每天时间很少。别鼓励我全都学，帮我选一个本周最该验证的方向。
```

## 7. 学后复盘

```text
我刚做完一个练习，但感觉只是照着答案完成了。请帮我判断这算不算掌握，并安排下一步。
```
