# focus-coach 示例 Prompt

## 多目标收敛

```text
我同时想学 TS、算法和多 Agent，但这个月只能重点投入一个。
```

## 最大约束

```text
我每天都在学，但两个月后仍然不会独立做项目，帮我判断最大问题。
```

## 是否继续

```text
这个方向已经投入很多，但和我的工作目标越来越远，要继续吗？
```

## 周期复盘

```text
根据我这周实际完成的内容，决定下周唯一主线。
```

## 配合 learning-coach

```text
先帮我决定这个月是否以 TypeScript 为主线；方向确定后，让 learning-coach 直接带我开始。
```
