# focus-coach 示例 Prompt

这些 prompt 用于展示 `focus-coach` 的典型触发方式。重点不是教学，而是帮助用户做取舍、收敛主线并定义下一步动作。

## 1. 多目标收敛

```text
我同时想学英语、写作、AI、剪辑和产品分析，但每天时间很少。别鼓励我全都学，帮我选一个本周最该验证的方向。
```

## 2. 优先级排序

```text
我现在有三个任务：准备面试、做个人项目、学 React。每个都重要，我不知道先做哪个。
```

## 3. 瓶颈诊断

```text
我最近一直很忙，但感觉没有进展。每天都在看资料、收藏工具、做计划。
```

## 4. 执行复盘

```text
我本周计划做完一个 demo，但实际只看了教程，没有写代码。帮我复盘并安排下一周。
```

## 5. 配合 learning-coach

```text
我想学 AI Agent，但不知道学到什么程度才算够。先用 focus-coach 帮我定义学习使命，再交给 learning-coach 设计第一个闭环。
```
