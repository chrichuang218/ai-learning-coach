# GLOSSARY.md 格式

`GLOSSARY.md` 是学习工作区的共同语言。它只记录用户已经能正确使用的术语，不是提前塞给用户的百科词典。

## 模板

```md
# {主题} Glossary

{1-2 句话说明这个术语表服务哪个学习主题。}

## Terms

**{术语}**：
{1-2 句话定义它是什么。}
_避免_：{容易混淆或不建议使用的说法}

**{术语}**：
{1-2 句话定义它是什么。}
_避免_：{容易混淆或不建议使用的说法}
```

## 规则

- 只有当用户已经能正确使用某个术语时，才加入术语表。
- 定义要短，只写“它是什么”，不要写成长教程。
- 多个说法冲突时，选择一个标准说法，把其他说法放进 `_避免_`。
- 后续解释、练习和学习记录要使用术语表里的标准语言。
- 当理解变深或原定义不准时，直接更新定义，不保留过时说法。
