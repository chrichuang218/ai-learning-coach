# 冒烟验证入口

两个 Skill 必须分开验证：

- [learning-coach-smoke-prompts.md](learning-coach-smoke-prompts.md)：现场带学、真实项目、渐进披露、连续答疑、证据沉淀和进度更新。
- [cross-domain-smoke-prompts.md](cross-domain-smoke-prompts.md)：语言、考试、写作、研究和职业技能的跨领域适配。
- [focus-coach-smoke-prompts.md](focus-coach-smoke-prompts.md)：目标取舍、最大约束、周期承诺和战略交接。
- [release-checklist.md](release-checklist.md)：发布前的静态、安装和行为验收记录。

关键路由：

```text
开始/继续/概念/Debug/学完了/日常进度 -> learning-coach
主线/优先级/是否继续/周度或阶段复盘 -> focus-coach
```
