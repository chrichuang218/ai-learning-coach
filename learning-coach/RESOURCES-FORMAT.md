# RESOURCES.md 格式

`RESOURCES.md` 记录高质量资源和真实学习材料。资源必须服务当前学习使命和下一步动作。

## 模板

```md
# Resources: {学习主题}

## Primary

- [{资源名称}]({链接或位置})
  {为什么可信，适合解决什么问题，当前该怎么用。}

## Practice Anchors

- {真实任务、项目、题库、作品、场景、论文、数据集、素材或练习入口}

## Communities

- {可获得真实反馈的社区、同伴、老师、论坛或线下场景}

## Parking Lot

- {以后可能有用但当前不追的资源。不要让它们干扰下一步}
```

## 规则

- 资源越少越好。优先一个高质量资源，而不是一串链接。
- 每个资源都要说明服务哪个动作。
- 不把收藏资源当作学习进展。
- 如果资源没有被使用、验证或复盘，不要把它写成核心材料。
